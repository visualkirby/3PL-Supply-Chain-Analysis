# Data Preparation & Validation
- Microsoft Excel
  - Data cleaning and normalization
  - Date and numeric validation checks
  - KPI calculation verification
  - Pivot-based exploratory analysis

# Data Analysis & Visualization
- Tableau Public
  - Interactive KPI dashboard
  - Shipment mode, vendor, and geographic performance analysis
  - Cost vs reliability tradeoff visualization
  - Tableau Story for guided stakeholder walkthrough

# Documentation & Version Control
- GitHub
  - Data dictionary and KPI definitions
  - Analysis log for traceability
  - Project documentation and portfolio presentation

# Analytical Approach
- Data validated prior to analysis to ensure metric integrity
- KPIs governed with explicit inclusion and exclusion rules
- Analysis focused on decision-oriented insights rather than descriptive statistics
- Visuals designed for clarity, interpretability, and executive use

# Why These Tools Were Chosen
- Excel provided transparent validation and auditability
- Tableau enabled interactive exploration and stakeholder-ready visuals
- GitHub ensured documentation consistency and reproducibility
